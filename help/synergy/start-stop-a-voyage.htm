Start Voyage

The Start Voyage  under the Actions column on the Voyage List table allows users to activate an open voyage.  

Perform the following steps to activate an open voyage: 

  1. Under the Actions column, click .

    A pop-up appears, prompting the user to select the voyage start date and time. By default, the date field populates the current date. Click the date field to view the date and time picker.
  2. Select the required date and time from the date and time picker, respectively, and click Start to start the voyage.

    The application allows the user to select only the planned start date and the current date.  

    A success message appears on the Voyages page, indicating that the voyage started successfully. 

    The status of the selected voyage displays Active. 

    There can be only one active voyage at a time. The user must close the active voyage to activate a new voyage. The following error message appears when the user starts a new voyage before closing the existing active voyage. 

    The user can activate only the Open voyages for which at least one loadable plan is confirmed. Otherwise, the following error message appears when the user clicks the Start icon indicating that the selected voyage cannot be activated since no loadable study data is confirmed.
