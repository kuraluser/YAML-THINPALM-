Delete Port

Perform the following steps to delete an existing port from the Port tab:

  1. From the menu bar, click CARGO PLANNING > Discharge Study. 

    By default, the page displays the discharge studies created against the active voyage.

    To view and edit the discharge study created against another voyage, from the Voyage No drop-down, select the required option. 
  2. From the Discharge Study List table, select the required discharge study. 
    The Cargo Nomination tab appears. Click Port. 
  3. From the drop-down against the port you want to delete, select Delete. 
    A pop-up appears, prompting you to confirm the action. Click YES to proceed. 

The user cannot delete the first discharging port added during the cargo nomination stage in the confirmed loadable study.
