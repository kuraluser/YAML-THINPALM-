Delete Port

To delete an existing port from the PORTS tab, perform the following steps:

  1. Navigate to CARGO PLANNING > Discharge Study. 

    By default, the page displays the discharge studies created against the active voyage.
    (Image)
    To view and edit the discharge study created against another voyage, select the required option from the Voyage No drop-down list. 
    (Image)
  2. From the Discharge Study List table, select the required discharge study. 
    (Image)
    The CARGO NOMINATION tab appears. Go to the PORT tab. 
  3. From the drop-down list against the port you want to delete, select DELETE. 
  4. A pop-up appears prompting you to confirm the action. Click YES to proceed. 

It is not possible to delete the first discharging port added during the cargo nomination stage in the confirmed loadable study.
