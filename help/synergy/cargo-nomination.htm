Cargo Nomination

The Cargo Nomination tab displays the planned quantity of the cargo to be loaded at each port during the voyage. The overall loadable quantity of the cargo must be within the tolerance limit. 

The Cargo Nomination tab includes the following fields to capture the cargo details during the loadable study stage:

Field Description

Loadable Quantity

Displays the maximum quantity of cargo that can be loaded into the vessel.

Unit

From the Unit drop-down, select the required unit. The available options are:

  • BBLS 
  • MT
  • KL
Add Cargo

Click Add Cargo to add up to seven cargoes for the loading operation. For more information, see Add Cargo.

Commingle

Click Commingle to enter the details of commingling cargoes. For more information, see Commingle.

Rules

Click Rules to enable or disable the applicable rules listed against the selected loadable study. The user can also edit specific values against different rules. The rules are updated from the Manage Rules section  under Admin. 

First Discharge Port

From the drop-down, select the first port to discharge the cargo. This data is used to auto-calculate the cargo allocation in slop tanks while generating an optimal loadable plan. 

Perform the following steps to complete the cargo nomination stage:

  1. From the Loadable Study page, select the required loadable study and navigate to the respective Cargo Nomination tab. 
  2. Click Add Cargo to add the cargo details. It is possible to add up to seven cargoes. For more information, see Add Cargo.
  3. Click Commingle to add the commingle cargo details, if applicable. 

    The Commingle Cargo pop-up appears. Enter the details in the corresponding fields and click Save to save the commingle cargo details. 

    For more information, see Commingle.
  4. From the First Discharge Port drop-down, select the first discharge port. 

    The Cargo to discharge first field appears.
  5. From the Cargo to discharge first drop-down, select the first cargo to discharge at the selected port if required. 

    The status of Cargo Nomination displays COMPLETE.

    The user can proceed to the Port tab only when the status of Cargo Nomination displays COMPLETE.
