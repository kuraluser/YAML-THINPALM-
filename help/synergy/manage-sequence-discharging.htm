Manage Sequence

Manage Sequence allows the user to configure the order of cargo discharging.

The section provides the following details:

Field Description

Add initial delay

Select an initial delay if required before starting the cargo discharging operation. To know the steps for adding an initial delay, see Add Initial Delay.

Add

Click Add to add the cargoes. If multiple cargoes are available, the user can arrange the cargo discharging sequence. To know the steps for adding a cargo, see Add Cargo.
