Stop Voyage

The Stop Voyage under the Actions column on the Voyage List table allows the user to deactivate an active voyage.  

Perform the following steps to stop an active voyage:

  1. Under the Actions column, click .

    A pop-up appears, prompting the user to select the voyage end date and time. By default, the date field populates the planned end date.
  2. Click the date field to view the date and time picker.
  3. Select the required date and time from the date and time picker, respectively, and click Stop to stop the voyage.

    The application allows the user to select only the dates between the planned end date and the current date. 

A pop-up appears, prompting the user to confirm that all the actual values are updated before closure.

  1. Click Yes to proceed. 

    The application allows the user to close a voyage only after updating the actual and BL values of the voyage against all ports.

    A success message appears on the Voyages page, indicating that the voyage stopped successfully.  

    The status of the selected voyage displays Closed.

     The user cannot edit a Closed voyage.
