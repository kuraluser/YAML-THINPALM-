Create Loadable Study

The Create New Loadable Study option allows the user to create a new loadable study against a voyage based on the cargo enquiry received.

Perform the following steps to create a loadable study:

  1. From the menu bar, click CARGO PLANNING > Loadable Study.

    By default, the page displays the Loadable Study List corresponding to the active voyage.

    The Create New Loadable Study option is disabled for an active voyage since the loadable study for the voyage is already confirmed. 

    To view the loadable study list created against another voyage, from the Voyage No drop-down, select the required option. 
  2. Click Create New Loadable Study.
    The New Loadable Study pop-up appears.
  3. The pop-up displays the following details and the user can edit the fields if required.
    Field Type Mandatory Description

    Duplicate Existing

    Drop-down

    No

    Create a loadable study by duplicating an existing loadable study. 

    New Loadable Study Name

    Text

    Yes

    Enter a name for the loadable study. The loadable study name is unique for every loadable study. 

    Enquiry Details

    Text

    No

    Add the enquiry details received. 

    Attach Mail

    Attachment

    No

    Attach the files related to the enquiries received for the respective voyage. 

    Charterer

    Text

    Yes

    Displays the current charterer name by default. 

    Sub Charterer

    Text

    No

    Enter the sub charterer name. 

    Loadline

    Drop-down

    Yes

    Enter the loadline data based on different voyage zones. By default, the field populates Summer. The user can select the required option from the drop-down. The available options are:

    • Summer
    • Winter
    • Tropical
    • Tropical to Summer
    • Tropical to Winter
    • Summer to Winter
    • Summer to Tropical
    • Winter to Summer
    • Winter to Tropical

    The loadable quantity is updated based on the loadline and draft changes. 

    Draft Mark

    Drop-down

    Yes

    Displays the draft mark corresponding to the loadline selected. The loadable quantity is updated based on the loadline and draft changes. 

    Draft Restriction

    Numeric

    No

    Enter the draft restrictions, if any. 

    Max Air Tem. in Voyage

    Numeric

    No

    Enter the maximum air temperature in the voyage. 

    Max Water Tem. in Voyage

    Numeric

    No

    Enter the maximum water temperature in the voyage. 
  4. Click Save to save the loadable study details.

    The respective Cargo Nomination tab appears, displaying a success message indicating that the loadable study is successfully created.
