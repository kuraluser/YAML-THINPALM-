Discharge Study Status

There are five different options to indicate the current status of a discharge study.  The current status of the discharge study determines the operations the user can perform against the respective discharge study. The available discharge operations are:

  • Edit discharge study
  • Duplicate discharge study
  • Delete discharge study

Each discharge study status is indicated by its respective symbol. The discharge study status options and the respective symbols are:

- Pending

- ALGO Processing Started

- Error Occured

- Plan Generated

- Confirmed

Pending - This status appears against the discharge studies for which the user has not completed all or any of the four discharge study stages including cargo nomination, port rotation, on hand quantities, and discharge study.

 

The user is allowed to initiate the discharge plan generation only when all the four discharge study stages are completed. The discharge operations allowed in this case are edit, delete, and duplicate.

ALGO Processing Started - This status appears against the discharge studies for which the user has completed all four cargo planning stages and initiated the discharge plan generation process.

 

No discharge operations are allowed in this stage. 

Error Occured - This status appears against the discharge studies for which Loadicator confronts errors during the algo processing stage.

 

The View Error option on the discharge study page allows the user to view the errors occured during the algo processing and edit the respective discharge study accordingly.

 

The discharge operations allowed in this stage are Edit, Delete, and Duplicate. 

Plan Generated - This status appears against the discharge studies for which the discharge plan is generated.

 

The View Plan option allows the user to view the discharge plan generated against the respective discharge study. The only discharge operation allowed in this stage is Duplicate. 

Confirmed - This status appears against the discharge studies for which a discharge plan is confirmed. 

 

The View Plan option allows the user to view the discharge plan confirmed against the discharge study. No discharge operations are allowed in this stage. 

For more information, see Discharge Study Operations.
