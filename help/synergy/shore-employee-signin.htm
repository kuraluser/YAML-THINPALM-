Shore User Sign In

Shore users can access the CP-DSS application either through KEYCLOAK or a Microsoft account.

Prerequisites

Consider the following prerequisites before attempting to sign in to the shore side CP-DSS application. 

  • Users must be registered in the Keycloak directory.

    Only the users with Admin privilege can add users to the Keycloack directory.

  • Users must have a Microsoft account. Once the user accesses the CP-DSS application through the Microsoft account, the user's login credentials are automatically updated in the Keycloak registry allowing the user to sign in to the CP-DSS application through KEYCLOAK from the next time. 

Perform the following steps to sign in to the CP-DSS application:

  1. Open a web browser and enter the following URL: https://cpdss-dev.alphaorimarine.com/login. 
  2. Click KEYCLOAK if you have already registered in the directory. Else, click MICROSOFT.
     The shore sign-in page appears. Enter the username and password in the respective fields. 
  3. Click Sign In.

    The Fleet page appears, displaying the details of all the vessels or the fleet managed by the user.

    For more information, see Shore Dashboard (Fleet). 

    A shore user can access the details of all the vessels that come under the fleet managed by the user.
