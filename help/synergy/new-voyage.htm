Create New Voyage

The Create New Voyage option allows the user to create a voyage for the future. The user can create a voyage from the vessel Dashboard and Voyages pages. Perform the following steps to create a voyage:

  1. Click New Voyage.
    The New Voyage pop-up appears, displaying the names of the Vessel, Captain, and the Chief Officer.

  2. The pop-up displays the following details and the user can edit the fields if required.

     
    Field Name Type Mandatory Description

    Voyage No

    		 Text

    Yes

    Enter a voyage number for the new voyage. The Voyage No is unique for each voyage.  

    Planned Start Date

    Date Picker

    No

    Enter the voyage start date.

    Start Date Timezone

    Drop-down 

    No

    Select a start date timezone for the planned voyage.

    Planned End Date

    Date Picker

    No

    Enter the voyage end date.

    End Date Timezone

    Drop-down 

    No

    Select an end date timezone for the planned voyage.

  3. Click Save to save the voyage details. 

    The Loadable Study List corresponding to the new voyage appears displaying a success message indicating the voyage is successfully created. 

    To create a loadable study against the voyage, click Create New Loadable Study. For more information, see Create Loadable Study.

    The Create New Loadable Study option is not available for the active voyage.
