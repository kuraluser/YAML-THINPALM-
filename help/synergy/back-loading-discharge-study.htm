Back Loading

Back Loading is the process of loading cargo to the selected tanks at a discharging port after the discharging operation. 

The Enable Back Loading check-box on the Discharge Study page allows the user to select whether to backload cargo at a particular discharging port. 

Perform the following steps to perform back loading at a discharging port: 

  1. From the menu bar, click Cargo Planning > Discharge Study. The Discharge Study List page for the active voyage appears. 
  2. From the Discharge Study List, select the required option.
    Alternatively, click Create New Discharge Study to create a new discharge study. For more information, see Create Discharge Study. 
  3. Click Discharge Study. The tab displays the list of all the discharging ports involved in the active voyage. 
  4. Under the required discharging port section, select Enable Back Loading check-box to enable backloading at that particular port. 
    A new row appears below the selected discharging port section. 
  5. The table displays the following details and the user can edit the fields if required. 
    Field Description

    Color

    Select the cargo color.

    Cargo Name

    Select the name of the cargo.

    Abbreviation

    Displays the abbreviation mapped against the respective cargo. The user can edit the field if required.

    BBLS

    Displays the dischargeable cargo quantity at the selected port in BBLS.

    MT

    Displays the dischargeable cargo quantity at the selected port in MT.

    KL

    Displays the dischargeable cargo quantity at the selected port in KL. The user can edit the value if required. 

    API

    Displays the API value. The user can edit the value if required. 

    TEMP

    Displays the temperature. The user can edit the value if required. 
  6. Click Save to save the details.

    Alternatively, click the Delete icon to delete the backloading details. For more information, see Delete Back Loading.
    Click Add to add more back loading cargoes.
