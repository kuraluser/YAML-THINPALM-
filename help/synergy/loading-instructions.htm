Loading Instructions

The Loading Instruction tab displays a set of loading instructions on the generated loading plan that are followed in the future while using it. 

To complete the Loading Instruction stage, perform the steps given below:

  1. From the different categories of instructions provided, choose a category.
  2. Select the required instructions. Click Select All to select all instructions.
  3. Click  to add a new instruction under the same section.
  4. Click Add New Sub Section to add a new section of instructions.
  5. Click SAVE to save the changes.
    Once the changes are saved, the status of Loading Instruction becomes COMPLETE.

    Click ADD NEW INSTRUCTION to add a new instruction.
