Add Port

The Add Port option on the PORT tab allows the user to add ports for bunkering and transit operations during the loadable study stage. 

Perform the following steps to add ports:

  1. Navigate to CARGO PLANNING > Loadable Study. 

    By default, the page displays the loadable studies created against the current active voyage.

    To view and edit the loadable study created against another voyage, select the required option from the Voyage No drop-down list. 
  2. From the Loadable Study List table, select the required loadable study. 

    The CARGO NOMINATION tab appears. Go to the PORT tab.
  3. Click Add Port. 

    A new row appears at the end of the port list table on the page.

    Update the fields in the row as per the table given below. 
    Field Name Type Mandatory Description

    Port Order

    Numeric

    Yes

    This field auto-populates the port order based on the order the user adds a port.

    The user can change the port order by dragging and dropping the required port to another level. 

    Port

    Drop-down

    Yes

    This field allows the user to select the name of the port from the list. 

    Port Code

    Text

    Yes

    This field auto-populates the port code mapped against the selected port.

    Operations

    Drop-down

    Yes

    This field allows the user to select an operation to carry out at the selected port. 

    The two available options are Bunkering and Transit. 

    Water Density

    Numeric

    Yes

    This field allows the user to enter the density of water at the selected port. 

    The default value is 1.025.

    Max Draft

    Numeric

    Yes

    This field allows the user to enter the maximum draft value at the selected port. 
    The value must be greater than or equal to zero. 

    Max Air Draft

    Numeric

    Yes

    This field allows the user to enter the maximum air draft value at the selected port. 
    The value must be greater than or equal to zero. 

    ETA

    Date Picker

    No

    This field allows the user to select the expected date of arrival of the vessel at the selected port.

    ETD

    Date Picker

    No

    This field allows the user to select the expected date of departure of the vessel from the selected port. 
  4. From the Drop-down icon (), select SAVE to save the changes. 

    The discharge port must be the last port. For more details, see Port Rotation.
