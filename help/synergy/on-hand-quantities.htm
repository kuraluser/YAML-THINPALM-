On Hand Quantities (OHQ)

The OHQ (On Hand Quantity) is the quantity of consumables remaining on board in the vessel tanks at each port. The consumables include fuel oil, diesel oil, and freshwater. The weight or volume of the consumables are updated at each port during the arrival and departure. 

The On Hand Quantities tab allows the user to update the volume or weight of the on board consumables at each port during the arrival and departure.  

The start port auto-populates the consumables quantity data from the previous voyage's end port data.

The On Hand Quantities tab displays the port ribbon based on the ports the user added in the Port tab. The user must select each port and enter the values against the required tanks.

The status of the On Hand Quantities tab displays COMPLETE only if data is entered against at least one tank from each tank category against all the ports in the port rotation.

The table displays the following details:

Field Description

Selected Port

Displays the list of ports selected for the voyage. The data is auto-populated from the Port tab. 

Tank Type

Select the tank type. The available options are:

  • Fuel oil tank
  • Diesel oil tank
  • Freshwater tank

Tank

Displays the list of available tanks based on the tank type selection. 

Density

Enter the density of the selected fuel type in each tank.

Arrival Quantity

Enter the volume or weight of the cargoes at the selected port during the arrival condition. The data is saved in the Synoptical table of the corresponding voyage. By default, the values are auto-populated from the Arrival condition field on the Synoptical Table. The user can switch the unit of the cargo quantity by choosing the required option from the Unit drop-down. 

Departure Quantity

Enter the volume or weight of the cargoes at the selected port during the departure condition. The data is saved in the Synoptical table of the corresponding voyage. By default, the values are auto-populated from the Departure condition field on the Synoptical Table. The user can switch the unit of the cargo quantity by choosing the required option from the Unit drop-down. 

The right pane of the tab displays the tank layout of arrival and departure.

The tank layout does not display the lube oil tanks.

The tank layout highlights the tank details the user selects from the OHQ table. By default, the CP-DSS application displays the tank layout of the arrival condition. 

Select Departure to view the tank layout during the departure condition.
