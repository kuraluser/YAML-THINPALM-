Edit Loadable Study

Perform the following steps to edit an existing loadable study:

  1. From the menu bar, click CARGO PLANNING > Loadable Study.

    By default, the page displays the Loadable Study List corresponding to the active voyage.

    To view the loadable study list created against another voyage, from the Voyage No drop-down, select the required option. 
  2. Click   to edit the loadable study name. 

    The Loadable Study pop-up appears, displaying the existing loadable study details.
  3. The pop-up displays the following details and the user can edit the fields if required.
    Field Type Mandatory Description

    Duplicate Existing

    Drop-down

    No

    The drop-down displays the selected loadable study details.  From the drop-down, select the  required loadable study created against the voyage. 

    New Loadable Study Name

    Text

    Yes

    Enter a name for the loadable study. The loadable study name is unique for every loadable study. 

    Enquiry Details

    Text

    No

    Displays the enquiry details received against the selected loadable study. The user can edit the details if required. 

    Attach Mail

    Attachment

    No

    Displays the files related to the enquiries received for the respective voyage. 

    Charterer

    Text

    Yes

    Displays the current charterer name by default. 

    Sub Charterer

    Text

    No

    Displays the sub charterer name of the selected loadable study. The user can select another sub-charterer if required. 

    Loadline

    Drop-down

    Yes

    Displays the loadline data based on the selected loadable study.

    The user can edit the data if required. The loadable quantity is updated based on the loadline and draft changes. 

    Draft Mark

    Drop-down

    Yes

    Displays the draft mark corresponding to the selected loadline. The loadable quantity is updated based on the loadline and draft changes. 

    Draft Restriction

    Numeric

    No

    Displays the draft restrictions marked against the selected loadable study if any. The user can edit the data if required.

    Max Air Tem. in Voyage

    Numeric

    No

    Displays the maximum air temperature in the voyage marked against the selected loadable study data. The user can edit the data if required. 

    Max Water Tem. in Voyage

    Numeric

    No

    Displays the maximum water temperature in the voyage, which is marked against the selected loadable study data. The user can edit the data if required. 
  4. Click Update to update the loadable study details.

    The respective Cargo Nomination tab appears, displaying a success message indicating that the loadable study is successfully updated.
