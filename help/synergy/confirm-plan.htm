Confirm Loadable Plan

The Confirm Plan option on the Loadable Plan page allows the user to select a particular plan for the respective loadable study. Perform the following steps to confirm a loadable plan:

  1. On the Loadable Plan page of the required loadable study, click Confirm Plan. 
    The Port Rotation pop-up appears.
  2. The pop-up displays the following details and the user can edit the fields if required.
    Field Type Mandatory Description

    Port Order

    Numeric

    Yes

    Displays the port order selected for the voyage. The data is updated based on the user entries provided during the Port Rotation section of the respective loadable study. 

    Port

    Text

    Yes

    Displays the name of the port. 

    Port Code

    Text

    Yes

    Displays the unique code corresponding to each port. 

    Operations

    Text

    Yes

    Displays the type of operations to carry out at each port. The data is updated based on the user entries provided during the Port Rotation section of the respective loadable study. 

    LAY - CAN

    Date Picker

    Yes

    Select a date range for reaching each port.

    ETA

    Date and Time Picker

    Yes

    Select the expected date and time of arrival of the vessel at each port. ETA should be less than ETD. The ETA of a port must not overlap with the ETA of the previous port. 

    ETD

    Date and Time Picker

    Yes

    Select the expected date and time of departure of the vessel at each port. ETD should be greater than ETA.
  3. Click Save to update the Port Rotation details.
    A confirmation pop-up appears, prompting the user to confirm the action. Click Yes to proceed. 
    The Confirm Plan button on the Loadable Plan page displays Confirmed. The status is also updated on the Loadable Pattern page of the selected loadable study. 

The user can confirm only one loadable plan against a loadable study. If any other plans generated against the selected loadable study are already confirmed, the following pop-up appears informing the user that the action reverts the status of the previously confirmed loadable plan. Click Yes to proceed. 

It is not possible to confirm the loadable plan for a loadable study generated against the active voyage. If the voyage is active, the following error message appears when the user tries to confirm the respective loadable plan.
