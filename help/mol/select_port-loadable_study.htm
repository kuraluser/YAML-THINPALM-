Select Port

The Select Port field in the Add Cargo row allows the user to select a loading port and required quantity for the selected cargo. 

  1. From the Add Cargo row, click Select Port.

    A pop-up corresponding to the selected cargo appears. 
  2. From the Select Port drop-down list, select the required port name. The drop-down lists all the ports mapped against the selected cargo. The user can choose multiple loading ports if required. 
  3. Click Add to add the quantity of cargo to be loaded at the selected port. 
  4. Enter the quantity in the QUANTITY field. 

    Alternatively, click the Delete icon () to cancel the changes. 
  5. Click Save to save the changes. 

    The API and Temperature values against the selected cargo gets automatically reset to default when the user clicks Save.
