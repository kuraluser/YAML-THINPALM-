Cow

Cow (Crude Oil Wash) is washing the selected crude oil tanks at the discharging ports. The Cow option on the Discharge Study tab allows the user to select the tanks and percentage of crude oil washing.

Perform the following steps to perform Cow at a discharging port:

  1. From the menu bar, click  Cargo Planning > Discharge Study. The Discharge Study List page for the active voyage appears. 
  2. From the Discharge Study List, select the required option.
    Alternatively, click Create New Discharge Study to create a new discharge study. For more information, see Create Discharge Study. 
  3. Click the Discharge Study tab. The tab displays the list of all the discharging ports involved in the active voyage. 

There are two modes of Cow:

  • Auto
  • Manual
Auto cow

For Auto Cow mode, the CP-DSS application automatically allocates the tanks for Cow based on the previous voyage history.  

Perform the following steps to perform Cow at a discharging port in the Auto mode:

  1. Under the required port name, from the Cow drop-down, select Auto.
  2. From the Percentage drop-down, select the percentage of Cow.
Manual cow

For Manual Cow mode, the user can select the tanks for Cow.

Perform the following steps to perform Cow at a discharging port in the Manual mode:

  1. Under the required port name, from the Cow drop-down, select Manual.
  2. From the Tanks drop-down, select the tanks for Cow.
