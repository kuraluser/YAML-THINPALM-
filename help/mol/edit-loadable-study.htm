Edit Loadable Study

Perform the following steps to edit an existing loadable study:

  1. From CARGO PLANNING, click Loadable Study.

    By default, the page displays the Loadable Study List corresponding to the current active voyage.

    To view the loadable study list created against another voyage, from the Voyage No drop-down, select the required option. 
  2. Click   to edit the loadable study name. 
    The LOADABLE STUDY pop-up appears displaying the existing loadable study details.
  3. Update the fields in the pop-up as per the table given below. 
    Field Type Mandatory Description

    Duplicate Existing

    Drop-down

    No

    This field populates the selected loadable study details.

    The user can choose the required loadable study created against the voyage from the drop-down. 

    New Loadable Study Name

    Text

    Yes

    This field allows the user to enter a name for the loadable study. 

    The loadable study name is unique for every loadable study. 

    Enquiry Details

    Text

    No

    This field populates the enquiry details received against the selected loadable study.

    The user can edit the details if required. 

    Attach Mail

    Attachment

    No

    This field populates the files related to the enquiries received for the respective voyage. 

    Charterer

    Text

    Yes

    The field populates the current charterer name by default. 

    Sub Charterer

    Text

    No

    This field populates the sub charterer name of the selected loadable study.

    The user can select another sub-charterer if required. 

    Loadline

    Drop-down

    Yes

    This field populates the loadline data based on the selected loadable study.

    The user can edit the data if required.  

    The loadable quantity is updated based on the loadline and draft changes. 

    Draft Mark

    Drop-down

    Yes

    This field populates the draft mark corresponding to the selected loadline. 

    The loadable quantity is updated based on the loadline and draft changes. 

    Draft Restriction

    Numeric

    No

    This field populates the draft restrictions marked against the selected loadable study, if any. 

    The user can edit the data if required.

    Max Air Tem. in Voyage

    Numeric

    No

    This field populates the maximum air temperature in the voyage marked against the selected loadable study data. 

    The user can edit the data if required. 

    Max Water Tem. in Voyage

    Numeric

    No

    This field populates the maximum water temperature in the voyage marked against the selected loadable study data.

     The user can edit the data if required. 
  4. Click Update to update the loadable study details.

    The respective CARGO NOMINATION tab appears displaying a success message indicating that the loadable study is successfully updated.
