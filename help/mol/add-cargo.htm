Add Cargo

The Add Cargo option on the Loadable Study page allows the user to add a maximum of seven cargoes for loading operations during the cargo nomination stage. 

Perform the following steps to add cargo for loading:

  1. From the menu bar, click CARGO PLANNING > Loadable Study. 

    By default, the page displays the loadable studies created against the active voyage.

    To view and edit the loadable study created against another voyage, from the Voyage No drop-down, select the required option. 
  2. Select the required loadable study from the Loadable Study List table. 

    The respective Cargo Nomination tab appears. 

    The application allows the user to search the cargo list table based on the field names. 

  3. Click Add Cargo. 

    Alternatively, duplicate an existing cargo if required. For more information, see Duplicate Cargo.
    A new row appears at the end of the cargo list table.
  4. The table displays the following details and the user can edit the fields if required. 
    Field Type Mandatory Description

    Priority 

    Drop-down

    Yes

    Add the cargo based on the priority. The user can select from a priority range of one to seven. 

    Color

    Color Picker

    Yes

    Select a color for the new cargo. The color of each cargo must be unique. This color is used to represent the respective cargo in the stowage plan. 

    Cargo

    Drop-down

    Yes

    Select the required cargo from the list.

    ABV

    Text

    Yes

    Add the abbreviation for the selected cargo. The abbreviation for each cargo must be unique. 

    Loading Port

    Pop-up

    Yes

    Select one or multiple loading ports for cargo loading and enter the loadable quantity of the selected cargo planned at each port.

    For more information, see Select Port. 

    Quantity

    Numeric

    Yes

    Displays the total quantity of cargo based on the data entered in the Loading Port field. 

    Maximum Tolerance

    Numeric

    Yes

    Enter the maximum tolerance limit for the selected cargo. The maximum tolerance must be less than or equal to 25. 

    Minimum Tolerance

    Numeric

    Yes

    Enter the minimum tolerance limit for the selected cargo. The minimum tolerance must be less than or equal to zero. 

    API EST

    Numeric

    Yes

    Enter the estimated API value for the cargo at the selected port. API value is the standard measurement used to calculate the density of cargo. 
    The application populates the default API value for the selected cargo. The user can edit the data if required. The API EST must be greater than or equal to eight.  
    Click  to view the API and Temperature history of the selected cargo. For more information, see API and Temperature History.

    TEMP EST

    Numeric

    Yes

    Enter the estimated temperature of the cargo. The application populates the default temperature of the selected cargo. The user can edit the data if required. 

    The TEMP EST must be greater than or equal to 40 degrees Fahrenheit. 

    Click  to view the API and Temperature history of the selected cargo.

    For more information, see API and Temperature History.

    Segregation 

    Drop-down

    Yes

    Select the segregation mode for cargo loading or discharging.   

    The available options are:

    • Single - Select this option if the loading or discharging uses a single valve pipeline. 
    • Double - Select this option if the loading or discharging operation uses a double valve pipeline. 
  5. Click  to save the changes.
