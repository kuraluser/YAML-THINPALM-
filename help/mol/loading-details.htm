Loading Details

Loading Details displays the various loading details of the voyage.

The section displays the following loading details:

Field Description

Sunrise/Sunset

Displays the sunrise and sunset time at the port. The time picker allows the user to set an appropriate time.

Start Time

Displays the expected start time of the loading operation.  The time picker allows the user to set an appropriate time.

Trim Allowed (Initial/Max/Final)  

Displays the initial, maximum, and final trim values. These are the allowed trim limits at a port at different stages, such as arrival and departure conditions, ballast stripping, and topping off. The user can edit the values if required.

Tide Details(HW/LW)

Enter the tide details at the port. For more information, see Add Tide Details.
