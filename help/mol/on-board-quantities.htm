On Board Quantities

OBQ (On Board Quantity) is the quantity of cargoes remaining in the tanks before starting a voyage. The volume and weight of OBQ is reduced from the overall capacity of the tanks while calculating the loadable cargo. This is applicable only for the first loading port and the value is auto-populated from the previous voyage history. 

The departure values of the last port are the arrival quantities on board at the first port in the new voyage. 

The ON BOARD QUANTITIES tab displays the details of the cargoes remaining in the tanks before starting the current voyage. 

The ON BOARD QUANTITIES tab includes the following fields to display the on board quantity details during the loadable study stage:

Field Description

Tank

This field displays the list of tanks in the vessel.

Cargo

This field auto-populates the cargo type from the previous or current voyage history.

API

This field allows the user to enter the API value of the remaining cargoes.

By default, the CP-DSS application auto-populates the API values from the previous voyage history.

Quantity

This field allows the user to enter the weight or volume of the remaining cargoes on board. 

By default, the quantity is displayed as zero.

The user can switch the unit of the cargo quantity by choosing the required option from the Unit drop-down list. 

The TANK FILLS section displays the tank layout and highlights the details of the tank the user selects from the OBQ table. 

To enable the load on top property for slop tanks, select the Enable Load on top for Slop Tanks check box.
