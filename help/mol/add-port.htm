Add Port

The Add Port option on the Port tab allows the user to add ports for bunkering  and transit operations during the Loadable Study stage. 

Perform the following steps to add ports:

  1. From the menu bar, click CARGO PLANNING > Loadable Study. 

    By default, the page displays the loadable studies created against the active voyage.

    To view and edit the loadable study created against another voyage, from the Voyage No drop-down, select the required option. 
  2. From the Loadable Study List table, select the required loadable study. 

    The Cargo Nomination tab appears. Click Port.
  3. Click Add Port. 

    A new row appears on the page at the end of the port list table.

    The table displays the following details and the user can edit the fields if required. 
    Field Type Mandatory Description

    Port Order

    Numeric

    Yes

    Displays the port order based on the order the user adds a port. Drag and drop the port to change the order.

    Port

    Drop-down

    Yes

    Select the name of the port from the list. 

    Port Code

    Text

    Yes

    Displays the port code mapped against the selected port.

    Operations

    Drop-down

    Yes

    Select an operation to carry out at the selected port. The two available options are:

    • Bunkering
    • Transit. 

    Water Density

    Numeric

    Yes

    Enter the density of water at the selected port. The default value is 1.025.

    Max Draft

    Numeric

    Yes

    Enter the maximum draft value at the selected port. The value must be greater than or equal to zero. 

    Max Air Draft

    Numeric

    Yes

    Enter the maximum air draft value at the selected port. The value must be greater than or equal to zero. 

    ETA

    Date Picker

    No

    Select the expected date of arrival of the vessel at the selected port.

    ETD

    Date Picker

    No

    Select the expected date of departure of the vessel from the selected port. 
  4. Click  > Save to save the changes. 

    While arranging the port order, the discharge port must be the last port. For more information, see Port Rotation.
