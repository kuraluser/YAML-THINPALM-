Discharging Sequence

Discharging Sequence displays the sequence of discharging of each cargo. The algorithm provides the estimated discharging sequence according to the details provided in the Discharging Information tab. The various Gantt charts available under Discharging Sequence helps the user to monitor the progress of the discharging operation. It displays the following details: 

  • Cargo sequence chart
  • Cargo pump chart
  • Ballast sequence chart
  • Ballast pump chart
  • Flow rate chart  
  • Stability parameters

Cargo Sequence Chart

The cargo sequence chart displays the cargo discharging rate of the active voyage. The X-axis represents the time in hours, and Y-axis represents the cargo name.

The available details are:

Field Description

Tank No

Displays the tank number.

Cargo Grade

Displays the cargo name.

Ullage

Displays the ullage value. Ullage is the void space in the tank measured from the top of the tank to the upper surface of the fluid. For each cargo, ullage values are marked on the graph in the cargo’s color.

Qty

Displays the cargo quantity in BBLS.

The vertical lines on the graph display each stage. Mouse hover on the ullage line to view the Hours, Starting Time, Ending Time, Quantity, and Ullage value. 

The COW details are displayed on the graph. Mouse hover on the COW icon to view the  COW Angle, Starting Time and Ending Time.

The legends used to indicate different statuses of cargo sequence chart are:

 - Full wash COW

 - Top wash COW

 - Bottom wash COW

 - Stripping by Educter

The graph also displays the total cargo quantity and loading rate of each stage at the bottom.

Cargo Pump Chart

The cargo pump chart displays the various cargo pumps and their corresponding pumping rate. The X-axis represents the time in hours, and Y-axis represents the pump name.  Various color lines marked on the graph represent the pumping rate of different cargo pumps.
The vertical lines on the graph display each stage. Mouse hover on the color lines to view the pumping rate. 

Ballast Sequence Chart

The ballast sequence chart displays the ballast discharging rate of the active voyage. The X-axis represents the time in hours, and Y-axis represents the tank. The blue lines marked on the graph represent sounding.

Sounding is the measured height of fluid from the surface of the fluid to the bottom of the tank. 

The vertical lines on the graph display each stage. Mouse hover on the ullage line to view Hours, Starting Time, Ending Time, Quantity, and Sounding.

Ballast Pump Chart

The ballast pump chart displays the various ballast pumps and their corresponding pumping rate. The X-axis represents the time in hours, and Y-axis represents the pump name.  Various color lines marked on the graph represent the pumping rate of different ballast pumps.
The vertical lines on the graph display each stage. Mouse hover on the color lines to view the pumping rate. 

Flow Rate Chart

The flow rate chart displays the flow rate of cargo.
The X-axis displays the flow rate in bbls, and the y-axis displays the time in hours. Mouse hover on the graph to view the aggregate values.

Stability Parameters

The stability parameters display the stability parameter values against time for different stages. They are:

Field Description

Fore Draft

Displays the fore draft value against time.

Aft draft

Displays the after draft value against time.

Trim

Displays the trim value against time.

 UKC

Displays the UKC (Under-keel Clearance) value against time.   

GM

Displays the GM (metacentric height) value against time.

Max Shearing Force

Displays the maximum shearing force against time.

 Max Bending Force

Displays the maximum bending force against time.

Mouse hover on the table to view the stability parameters at that point.
