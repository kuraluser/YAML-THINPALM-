Berth Details

Berth Details displays the various berth details of the voyage. The berth information is available after communicating with the terminal operator of the discharge port and it is mentioned in the terminal agreement. The terminal operators provide the details for discharging such as the berth allocated, the hose connection, the maximum loading rate and so on.
Perform the following steps to add the berth details:

  1. Click .
    The Select a berth drop-down appears.
    (img)
  2. Select the required berth from the list.
    The fields Name, Depth(M), Manifold Height, Max Manifold Pressure, Sea Draft Limitation(M), Air Draft Limitation(M), and Line Displacement(BBLS) auto-populates once a berth is selected.
  3. To change the selected berth and add a new berth, click  and repeat step 2.

    If the user adds multiple berths under Berth Details, all fields auto-populate the values of the latest selected berth.

           To select any previously selected berth, click that berth.

The berth details section displays the following details:

Field Description

Name

Displays the berth name.

Depth(M)

Displays the depth value in meter

Manifold Height

Displays the max height of the manifold pipe in the vessel that can be berthed at the terminal.

Max Manifold Pressure

 Displays the allowed maximum pressure at the manifold.

Alongside, Hose Connection

Enter the hose connection for discharging operation.

Sea Draft Limitation(M)

Displays the sea draft limitation value.

Air Draft Limitation(M)

 Displays the air draft limitation value.

Cargo Circulation, Air Purge

Select the check-box to enable cargo circulation between tanks before the discharging operation.

Line Displacement(Bbls)

Enter the line displacement value. The value shows the amount of residual cargo that remained in the pipeline. Before filling the new cargo, this residual amount needs to be cleaned from the pipeline.

Special Regulation & Restriction

Enter any special regulation and restriction message received from the terminal.

Items to be agreed with Terminal

Enter the items that are agreed with the terminal.
