Edit Discharge Study

To edit an existing discharge study, perform the steps given below:

  1. Navigate to CARGO PLANNING > Discharge Study.

    The Discharge Study List corresponding to the current active voyage appears by default. 

    To view the discharge study list created against another voyage, select the required option from the Voyage No drop-down list. 
  2. Click the drop-down icon () against the discharge study name you want to edit. 
  3. Click EDIT. 
  4. The EDIT DISCHARGE STUDY pop-up appears displaying the existing discharge study details.

    Update the fields in the pop-up as per the table given below. 
    Field Name Type Mandatory Description
    New Discharge Study Name Text Yes

    This field populates the name of the selected discharge study. The user must  edit the fielf and enter a name for the new discharge study. 

    The discharge study name is unique for every discharge study. 
    Enquiry Details Text No This field populates the enquiry details received against the selected discharge study. The user can edit the data if required.  
  5. Click Update to update the discharge study details.

    The respective CARGO NOMINATION tab appears displaying a success message indicating the discharge study is successfully updated.
