Loadable Study Details

The Loadable Study phase consists of four different sections:

  • Cargo nomination - Displays the planned quantity of cargo to be loaded at each port during the voyage. For more information, see Cargo Nomination.
  • Port rotation - Displays the details of all the ports included in the selected voyage. For more information, see Port Rotation.
  • On Hand Quantities (OHQ) - Displays the cargo remaining in the vessel tanks at each port. For more information, see On Hand Quantities (OHQ).
  • On Board Quantities (OBQ) - Displays the cargo remaining in the tanks before starting a voyage. For more information, see On Board Quantities (OBQ). 

The following options are common for all the loadable study sections:

Voyage - The Voyage drop-down allows the user to select and view the loadable study details of another voyage. 

Loadable Study List- The left pane on the Loadable Study page lists the loadable studies generated against the corresponding voyage and its status, and the applicable loadable operations.

Select a loadable study name from the list to navigate to the respective Cargo Nomination tab. 

The search and sort options allow the user to search the loadable study list based on the loadable study names and filter the loadable study list view.

To create a new loadable study against the voyage, click New.

To access the corresponding Loadable Study List page, click View Detail. 

To collapse and expand the  Loadable Study List view, click   and . 

Loadable Quantity - The Loadable Quantity field displays the planned loadable quantity for the selected voyage. 

Click the loadable quantity value to view the details and edit the loadable quantity if required. 

Rules - The Rules option allows the user to enable or disable the applicable rules listed against the selected loadable study. 

The user can also edit specific values against different rules. 

Click Save Changes to update the rule settings. 

Pattern History - The Pattern History option allows users to view all the loadable patterns generated against the respective loadable study. This option is available for the loadable studies that have generated at least one pattern.
