Discharging Information

The Discharging Information tab displays the various vessel parameters and cargo details of the active voyage, specific to the discharge process. The values are auto-populated as specified in the confirmed Discharge Study Plan generated in the Cargo Planning section. 

The user can update every field’s data based on the discussion with the terminal and as per the terminal agreement.

Perform the following steps to complete the Discharging Information tab:

  1. From the menu bar, click Operations. Select the specific discharging port from the port ribbon. 
  2. Edit the fields of the following sections if required:
    • Discharging Details – Displays the discharging details of the active voyage. For more information, see Discharging Details.
    • Discharging Rates – Displays the discharging rates of the vessel. For more information, see Discharging Rates.
    • Berth Details – Displays the berth details of the voyage. For more information, see Berth Details.
    • Machinery Available – Displays the cargo machinery in use. For more information, see Machinery Available.
    • Pumps Available – Displays the pumps in use. For more information, see Pumps Available.
    • Cargo Details – Displays the cargo details of the vessel. For more information, see Cargo Details.
    • Cargo to be Discharged - Displays the details of the cargo that are going to be discharged as per plan. For more information, see Cargo to be Discharged.
    • No. of Stages – Displays the number of stages where the user can define the number of stages required at a discharging port. For more information, see No. of Stages.
    • Manage Sequence- Displays the cargo discharging order where the user can define the order. For more information, see Manage Sequence.
    • COW Plan and COW options - Displays the COW options and COW types. For more information, see COW Plan and COW Options.
    • Time for Post Discharge Stage - Displays the time for dry check, slop discharging, final stripping, and fresh oil washing. For more information, see Time for Post Discharge Stage.
  3. Click SAVE to save the changes.
    The status of Discharging Information displays COMPLETE.
