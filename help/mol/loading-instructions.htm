Loading Instructions

The Loading Instructions tab displays a set of loading instructions on the generated loading plan so that while executing the plan, the terminal operators can consider these instructions for safe and efficient operation. The loading instructions are created in Admin > Manage Instructions. 

Perform the following steps to complete the Loading Instructions tab:

  1. From the menu bar, click Operations. Select the specific loading port from the port ribbon. By default, the application displays the Loading Information tab. To access the loading instructions, click Loading Instructions tab.
  2. From the different categories of instructions provided, choose a category.
  3. Select the required instructions. Click Select All to select all instructions.
  4. Click Add New Sub Section to add a new section of instructions.
  5. Click  to add a new instruction under the subsection.
    Click ADD NEW INSTRUCTION to add a new instruction.
  6. Click SAVE to save the changes.
    The status of Loading Instructions displays COMPLETE.

The user can proceed to Generate Sequence and Plan only when the status of Loading Information and Loading Instructions displays COMPLETE.
