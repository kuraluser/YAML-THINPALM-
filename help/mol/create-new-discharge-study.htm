Create Discharge Study

The Create New Discharge Study option allows users to create a new discharge study against the active voyage. The user can create a discharge study against a voyage only when the loading operation is complete, and the B/L and actual values are updated in the Synoptical Table of the respective voyage. 

Once the user creates Discharge Study against the active voyage, all the loading data fields in the synoptical table of the voyage get disabled. The user can create multiple discharge studies against the active voyage. All the discharge studies get displayed under the Discharge Study List of the respective voyage. 

Perform the following steps to create a new discharge study:

  1. From the menu bar, click CARGO PLANNING > Discharge Study.

    By default, the page displays the Discharge Study List corresponding to the active voyage.

    To view the discharge study list created against another voyage, from the Voyage No drop-down, select the required option. 
  2. Click Create New Discharge Study.
    The New Discharge Study pop-up appears.
  3. The pop-up displays the following details and the user can edit the fields if required.
    Field Type Mandatory Description

    New Discharge Study Name

    		 Text Yes

    Enter a name for the new discharge study. The discharge study name is unique for every discharge study. 

    Enquiry Details

    		 Text No

    Enter the enquiry details received. 
  4. Click Save to save the discharge study details.

    The respective Cargo Nomination tab appears, displaying a success message indicating the discharge study is successfully created. 

    If the actual and B/L figure details are not updated in the Synoptical Table of the corresponding voyage, the user cannot create a new discharge study. The following error message appears on the page when the user clicks the Save button.
