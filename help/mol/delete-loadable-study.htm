Delete Loadable Study

To delete an existing loadable study, perform the following steps:

  1. Navigate to CARGO PLANNING > Loadable Study.

    By default, th page displays the Loadable Study List corresponding to the current active voyage.

    To view the loadable study list created against another voyage, select the required option from the Voyage No drop-down list.
  2. Click the drop-down icon () against the loadable study name you want to delete.
  3. Click DELETE.
  4.  A pop-up appears prompting the user to confirm the action. Click Yes to proceed. 

    The Loadable Study page appears displaying a success message indicating the loadable study is successfully deleted.
