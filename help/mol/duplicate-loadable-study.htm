Duplicate Loadable Study

Perform the following steps to duplicate an existing loadable study:

  1. From the menu bar, click CARGO PLANNING > Loadable Study.

    By default, the page displays the Loadable Study List corresponding to the active voyage.

    To view the loadable study list created against another voyage, from the Voyage No drop-down, select the required option. 
  2. Click  > Duplicate to duplicate the loadable study.

    The New Loadable Study pop-up appears displaying the existing loadable study details.
  3. The pop-up displays the following details and the user can edit the fields if required.
    Field Type Mandatory Description

    Duplicate Existing

    Drop-down

    No

    Displays the name of the selected loadable study. 

    New Loadable Study Name

    Text

    Yes

    Displays the name of the selected loadable study. The user must edit the loadable study name. The loadable study name is unique for every loadable study. 

    Enquiry Details

    Text

    No

    Displays the enquiry details received against the selected loadable study. The user can edit the details, if required. 

    Attach Mail

    Attachment

    No

    Displays the attached files related to the enquiries received for the respective  voyage. The user can edit the details if required.

    Charterer

    Text

    Yes

    Displays the current charterer name by default. 

    Sub Charterer

    Text

    No

    Displays the sub charterer's name entered for the selected loadable study. The user can edit the sub charterer name if required.  

    Loadline

    Drop-down

    Yes

    Displays the loadline mapped against the selected loadable study. The user can select the required option from the drop-down. 

    The available options are:

    • Summer
    • Winter
    • Tropical
    • Tropical to Summer
    • Tropical to Winter
    • Summer to Winter
    • Summer to Tropical
    • Winter to Summer
    • Winter to Tropical

    The loadable quantity is updated based on the loadline and draft changes.  

    Draft Mark

    Drop-down

    Yes

    Displays the draft mark corresponding to the selected loadline. The loadable quantity is updated based on the loadline and draft changes. 

    Draft Restriction

    Numeric

    No

    Displays the draft restrictions entered by the user against the selected loadable study. The user can edit the details if required.

    Max Air Tem. in Voyage

    Numeric

    No

    Displays the maximum air temperature in the voyage  entered by the user against the selected loadable study.  The user can edit the details  if required.

    Max Water Tem. in Voyage

    Numeric

    No

    Displays the maximum water temperature in the voyage  entered by the user against the selected loadable study.  The user can edit the details  if required.
  4. Click Save to save the loadable study details.

    The respective Cargo Nomination tab appears, displaying a success message indicating the loadable study is successfully created.
