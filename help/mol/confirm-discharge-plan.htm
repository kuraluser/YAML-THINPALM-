Confirm Discharge Study Plan

The Confirm Plan option on the Discharge Study Plan page allows the user to confirm the discharge study plan for the active voyage. The user can confirm only one discharge study plan against a voyage. 

Perform the following steps to confirm a discharge study plan for the active voyage:

  1. From the menu bar, click Cargo Planning > Discharge Study.
    By default, the CP-DSS application displays the discharge study list mapped against the active voyage.
  2. Select a discharge study against which the status is Plan Generated.

    The Confirm Plan option is available only for the Discharge Study for which the status is Plan Generated.

  3. On the Discharge Study tab, click View plan. 

    The respective Discharge Study Plan page appears.
  4. Click Confirm Plan.

    A pop-up appears, prompting the user to confirm the action.
  5. Click Yes to proceed. 

    The status of the respective discharge study displays Confirmed. 

After confirmation, the port details under the specific discharge study plan get listed to the Synoptical Table. Once a plan is confirmed, the user cannot make any changes.

If the user confirms another discharge study plan against the active voyage, the confirmed discharge study status is automatically reverted to Plan Generated.
