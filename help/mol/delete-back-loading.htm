Delete Back Loading

Perform the following steps to delete a cargo from the back loading list at a port. 

  1. Navigate to Cargo Planning > Discharge Study. The Discharge Study List page for current voyage appears. 
  2. From the Discharge Study List, select the required option.
    Alternatively. click Create New Discharge Study to create a new discharge study. For more information, see Create New Discharge Study. 
  3. Go to Discharge Study tab. The tab displays the list of all the discharging ports involved in the current voyage. 
  4. Under the required dicharging port section, from the back loading cargo list, select Delete against the required cargo name.
  5. A confirmation pop-up appears prompting the user to confirm the action. Click Yes to proceed. 

    The selected cargo gets deleted from the back loading list at the selected port.
