Add Cargo

The Add option allows the user to add cargoes under the Manage Sequence section. Perform the following steps to add a cargo:

  1. Click .
    A row appears in the table.
  2. Enter the following details in the row:
    • Cargo - Select the cargo name from the drop-down.
    • Quantity - Once cargo is selected, the field auto-populates cargo quantity.
    • Reason for Delay - Select the reason for delay from the drop-down.
    • Duration - Enter the delay duration.
