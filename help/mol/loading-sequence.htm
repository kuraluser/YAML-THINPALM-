Loading Sequence

ALGO provides the estimated Loading sequence according to the details provided in the Loading Information and Loading Instruction stages. The various Gantt charts available under Loading Sequence helps the user to monitor the progress of the Loading Operation. It displays the following details: 

  • Cargo loading sequence chart
  • Ballast loading sequence chart
  • Deballasting sequence chart
  • Flow rate chart  
  • Stability parameters

Cargo Loading Sequence Chart

The Cargo loading sequence chart displays the cargo loading rate of the active voyage. The X-axis represents the time in hours, and Y-axis represents the cargo name. The available details are:

Field Description

Tank No

Displays the tank number

Cargo Grade

Displays the cargo name

Ullage

Displays the ullage value. Ullage is the void space in the tank measured from the top of the tank to the upper surface of the fluid. For each cargo, ullage values are marked on the graph in the cargo’s color.

Qty

Displays the cargo Qty in BBLS

The vertical lines on the graph display each stage. Mouse hover anywhere on the ullage line to view the Time, Qty, and Ullage value. The graph also displays the Total cargo quantity and Loading Rate of each stage at the bottom.

Ballast Loading Sequence Chart

Ballast loading sequence chart displays the ballast loading rate of the active voyage. The X-axis represents the time in hours, and Y-axis represents the tank. The blue lines marked on the graph represent sounding.

Sounding is the measured height of fluid from the surface of the fluid to the bottom of the tank. 

The vertical lines on the graph display each stage. Mouse hover anywhere on the ullage line to view the Time, Qty, and Sounding value.
Stripping by Eductor- After ballasting, the remaining sediments (fuel residue) are sucked using eductor. 

Deballasting Sequence Chart

Deballasting sequence chart displays the deballasting loading rate of the active voyage. The X-axis represents the time in hours, and Y-axis represents the pump name.  Various color lines marked on the graph represent the sounding of different ballast pumps.
 The vertical lines on the graph display each stage. Mouse hover anywhere on the sounding line to view the deballasting rate. 

Flow Rate Chart

The flow Rate chart displays the flow rate of cargo.
The X-axis displays the flow rate in bbls, and the y-axis displays the time in hours. Mouse hover on the graph to view the aggregate values.

Stability Parameters

Stability Parameters displays the stability parameter values against time. They are:

Field Description

Fore Draft

Displays the fore draft value against time

Aft draft

Displays the air draft value against time

Trim

Displays the trim value against time

  UKC

Displays the UKC value against time.   

GM

Displays the GM value against time.

Max Shearing Force

Displays the maximum shearing force against time

  Max Bending Force

Displays the max bending force against time.

Mouse hover on the table to view the stability parameters at that point.
